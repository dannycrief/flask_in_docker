# flask_in_docker
<h2>Running program:</h2>
<p>All you need to do is:</p>
<ol>
    <li>Clone project</li>
    <li>Create venv</li>
    <li>Install requirements</li>
    <li>Run <b>sudo docker-compose up</b></li>
    <li>Open http://localhost:5000/*number*, where number is N in Fibonacci's sequence</li>
</ol>
